# 設定改進摘要

## 已完成的變更

### 1. 增強版 `requirements.txt`
- 加入了清晰的分類註解
- 針對平台的依賴處理：`gunicorn` 僅在 Linux/Unix 上安裝
- 記錄了每個依賴套件的用途
- 使用版本限制以確保穩定性

### 2. 改進的安裝腳本

#### Windows (`setup.bat`)
- 安裝後自動驗證套件
- 增強後續步驟引導
- 提供清楚的資料庫設定和 Ollama 安裝指引

#### Linux/Mac (`setup.sh`)
- 安裝後自動驗證套件
- 增強後續步驟引導
- 包含生產環境部署選項

### 3. 新增生產環境部署腳本

**`run_production.sh`** (僅限 Linux/Unix)
- 使用 Gunicorn 以獲得生產級效能
- 以背景守護行程 (Daemon) 方式執行服務
- 可設定 Worker 數量（基本服務 4 個，AI 服務 2 個）
- PID 檔案管理，便於服務控制

### 4. 環境驗證腳本

**`validate_setup.py`**
- 檢查 Python 版本相容性
- 驗證所有必要依賴是否已安裝
- 測試資料庫連線
- 檢查 Ollama 可用性（選用）
- 提供清楚的通過/失敗回饋

### 5. 更新文件

**README.md 增強：**
- 新增系統需求章節
- 記錄所有依賴及其版本
- 新增生產環境部署指南
- 包含環境驗證說明
- 針對不同平台的部署建議

## 依賴套件說明

所有依賴套件都有明確用途：

| 套件 | 版本 | 用途 | 理由 |
|---------|---------|---------|---------------|
| flask | 3.0.0+ | Web 框架 | 業界標準，輕量級 |
| flask-cors | 4.0.0+ | CORS 支援 | API 存取所需 |
| ollama | 0.1.0+ | AI 整合 | 核心 AI 功能 |
| sqlalchemy | 2.0.0+ | ORM | 型別安全的資料庫操作 |
| pymysql | 1.1.0+ | MySQL 驅動 | 純 Python，無 C 依賴 |
| pandas | 2.0.0+ | 資料處理 | Excel 匯入，資料轉換 |
| openpyxl | 3.1.0+ | Excel 讀取 | ISO 286 資料來源格式 |
| requests | 2.31.0+ | HTTP 客戶端 | API 測試，外部呼叫 |
| python-dotenv | 1.0.0+ | 設定管理 | 環境變數處理 |
| gunicorn | 21.0.0+ | 生產環境伺服器 | 生產級 WSGI (僅限 Linux) |

## 安裝可靠性

### 錯誤處理
- 兩個安裝腳本都會檢查 Python 是否可用
- 驗證虛擬環境建立
- 先升級 Pip 再安裝依賴
- 清晰的錯誤訊息與故障排除步驟

### 驗證
- 安裝後套件驗證
- 可選的 `validate_setup.py` 進行全面檢查
- 服務啟動前測試資料庫連線
