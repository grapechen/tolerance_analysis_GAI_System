# 離線版公差查詢系統

這是一個完全離線的 ISO 286 公差查詢系統，無需安裝任何軟體或資料庫。

## 特點

- ✅ 完全離線運作
- ✅ 無需安裝 Python 或資料庫
- ✅ 只需瀏覽器即可使用
- ✅ 包含完整的 ISO 286 資料（約 5000 筆）
- ✅ 支援所有查詢功能（IT 公差、軸公差、孔公差、配合分析）

## 使用方式

### 方法 1：直接開啟（推薦）

直接用瀏覽器開啟 `index.html` 文件即可使用。

### 方法 2：本地伺服器

如果瀏覽器有安全限制，可以使用簡單的 HTTP 伺服器：

**Python 3:**
```bash
cd client
python -m http.server 8000
```

然後訪問 http://localhost:8000

**Node.js:**
```bash
cd client
npx http-server
```

## 文件說明

- **index.html** - 網頁介面（約 20 KB）
- **tolerance_data.js** - 完整資料（約 4.2 MB）
  - 包含 IT 基本公差
  - 包含軸公差資料
  - 包含孔公差資料

## 適用場景

- 🏭 工廠現場無網路環境
- 💻 筆記型電腦離線使用
- 📱 行動裝置（可存在手機中）
- 🎓 教學展示
- 🔒 對資料隱私有極高要求的環境

## 與線上版本的差異

| 功能 | 離線版 | 線上版 |
|------|--------|--------|
| IT 基本公差查詢 | ✅ | ✅ |
| 軸公差查詢 | ✅ | ✅ |
| 孔公差查詢 | ✅ | ✅ |
| 配合分析 | ✅ | ✅ |
| AI 智能助手 | ❌ | ✅ |
| 需要安裝 | ❌ | ✅ Python + MySQL |
| 需要網路 | ❌ | ❌ |
| 資料更新 | 需手動更新 | 自動從資料庫讀取 |

## 更新資料

如果需要更新離線版本的資料，可以：

1. 在線上版本中更新資料庫
2. 執行資料匯出腳本（待開發）
3. 替換 `tolerance_data.js` 文件

## 技術細節

- 純 JavaScript 實作，無外部依賴
- 使用 Array.find() 進行資料查詢
- 資料格式：JSON
- 相容所有現代瀏覽器

## 限制

- 無 AI 智能助手功能
- 資料更新需要手動替換文件
- 檔案較大（4.2 MB），首次載入需要時間

## 建議

- 如果有網路和伺服器環境，建議使用線上版本
- 如果需要完全離線或極簡部署，使用此離線版本
- 可以將此目錄複製到 USB 隨身碟中隨身攜帶
