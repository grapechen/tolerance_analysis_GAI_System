# 公差配合功能描述查詢指南
### RAS-400 轉台軸承座資料庫

---

## 使用說明

根據零件的**使用情境或功能**，從下方找到對應的功能描述，即可查詢推薦的配合公差。

---

## 一、留隙配合（Clearance Fit）
> 孔比軸大，兩件之間保有間隙，適合需要相對運動或可拆卸的場合。

### 🔄 運轉和滑動（RC 系列）

| 功能描述 | 推薦配合 | 適用情境 |
|---|---|---|
| 精確定位 | H5/g4（RC1） | 精確滑動，幾乎無間隙 |
| 定溫環境運轉 | H6/g5（RC2） | 精確定位，輕度滑動 |
| 低速轉動、低軸頸壓力 | H7/f6（RC3） | 精密機械慢速運轉 |
| 中速轉動、中軸頸壓力 | H8/f7（RC4） | 一般機械中速運轉 |
| 溫差大、高速轉動、高軸頸壓力 | H8/e7（RC5） | 較高速或重負荷 |
| 溫差大、高速轉動、高軸頸壓力 | H9/e8（RC6） | 高速重負荷 |
| 溫差大、精度要求不高 | H9/d8（RC7） | 自由運轉，精度較低 |
| 大公差 | H10/c9（RC8） | 寬鬆商業公差 |
| 大公差 | H11/c11（RC9） | 最寬鬆配合 |

| 精確定位、小正間隙 | H7/g6（LC5） | 精確定位，最小正間隙 |
| 定位、中等間隙（如銷孔配合） | H9/f8（LC6） | 靜止定位，中等間隙 |

| 裝拆方便優先、鬆緊固件配合 | H11/c10（LC9） | 自由裝拆優先 |
| 裝拆方便優先、最鬆緊固件配合 | H12/b11（LC10） | 寬商業公差 |

## 二、過渡配合（Transition Fit）
> 可能有小間隙或小干涉，適合需要精確定位但仍可拆卸的場合。

### 📐 定位過渡（LT 系列）

| 功能描述 | 推薦配合 | 裝配方式 |
|---|---|---|
| 定位、可裝拆 | H7/j6（LT1） | 輕推即可 |
| 定位 | H8/j7（LT2） | 輕推即可 |
| 準確定位 | H7/k6（LT3） | 輕打入 |
| 準確定位 | H8/k7（LT4） | 壓入 |
| 準確定位（較緊） | H7/n6（LT5） | 中等壓力 |

### 🔩 RAS-400 專用（過渡）

| 功能描述 | 推薦配合 | 零件對應 |
|---|---|---|

| 工作台與工作台心軸精確定位、螺栓+定位銷固定 | H7/k6 | 工作台(1) ↔ 工作台心軸(5) |

## 三、過盈配合（Interference Fit）
> 軸比孔大，需壓入或熱脹冷縮裝配，固定後不易拆卸，適合永久接合或高扭矩傳遞。

### 📌 定位干涉（LN 系列）

| 功能描述 | 推薦配合 | 適用情境 |
|---|---|---|
| 剛性對準（無需傳遞大扭矩） | H7/p6（LN2） | 精確同軸剛性固定 |

### 💪 力配合（FN 系列）

| 功能描述 | 推薦配合 | 適用情境 |
|---|---|---|
| 壓力小、堅固 | H6/n5（FN1） | 輕壓入，薄件或鑄鐵件 |
| 壓力小、堅固 | H6/p5（FN1） | 輕壓入，薄件或鑄鐵件 |
| 鑄鐵件配合使用 | H7/s6（FN2） | 中等壓入 |
| 壓力大 | H7/t6（FN3） | 重壓入，鋼件 |
| 高應力 | H7/u6（FN4） | 強制壓入 |
| 高應力 | H8/x7（FN5） | 最大干涉 |

### 🔩 RAS-400 專用（過盈）

| 功能描述 | 推薦配合 | 零件對應 |
|---|---|---|

> **注意事項**
> - YRT100 為螺栓鎖附型轉台軸承，軸承內外圈均以螺栓固定，**不依賴過盈配合**傳力
> - 工作台心軸(5)為被動角色，依各介面零件選擇功能查詢
> - 馬達座(10)同上，為底部支撐結構，依各介面零件查詢
> - LC10（H12/b11）、LC11（H13/b12）軸偏差代號為近似值，不在 ABC 協議範圍，建議以 ANSI B4.1 原版確認
> - ANSI 英制系列（RC/LC/LT/FN）與 ISO 公制系列選用方式相同，請依圖面標準系統選擇
> - 資料來源：Machinery's Handbook 31st Edition、YRT100 軸承規格表

## 五、YRT100 軸承組裝相容性驗證

> 驗證目的：確認各推薦配合不會導致 YRT100 無法組裝。

### YRT100 關鍵尺寸（d=100 × D=185 × H=38 mm）

| 參數 | 值 | 說明 |
|---|---|---|
| D1 max | 160 mm | 軸承孔內側鄰接結構最大外徑（工作台心軸體） |
| J | 112 mm PCD | 內圈固定螺栓節圓（16 孔，M5）⚠ |
| J1 | 170 mm PCD | 外圈固定螺栓節圓（15 孔，M5）⚠ |
| MA | 8.5 Nm | 螺栓鎖緊扭力（DIN 912，等級 10.9） |
| 最高轉速 | 280 rpm | |

### 配合相容性結論

| 配合 | 尺寸 | 公差範圍 | 相容性 | 說明 |
|---|---|---|---|---|
| js5（工作台心軸內圈座） | d=100mm | ±7.5 μm | ✓ 相容 | 過渡配合，螺栓固定不依賴干涉；若有微量干涉請加熱內圈至 80–100°C 後安裝，禁止敲打 |
| H6（軸承座外圈孔） | D=185mm | +0/+46 μm | ✓ 相容 | 留隙配合，外圈可輕鬆置入後螺栓鎖附 |
| D1 max ≤ 160mm | 心軸體外徑 | 設計尺寸 | ✓ 無衝突 | 為設計幾何尺寸限制，非配合公差問題，需於心軸圖面確認 |
| 螺栓孔對準 | 工作台 16 孔 @ J=112mm | — | ✓ 無衝突 | 公差不影響螺栓孔位，但需確認工作台孔型與節圓一致 |
| 螺栓孔對準 | 軸承座 15 孔 @ J1=170mm | — | ✓ 無衝突 | 同上，軸承座孔型需與外圈節圓一致 |
| H7/u6（轉動軸壓入心軸） | S-4 位置 | 強制壓入 | ⚠ 設計注意 | 壓入可能造成鄰近軸承座面變形，建議：①先安裝軸承再壓入轉動軸，或②壓入後重新研磨軸承座；若軸承座與壓入區距離 > 1倍直徑則風險低 |

### 組裝順序建議

1. 加工工作台心軸（js5 軸承座面、S-4 轉動軸壓入孔同時加工至最終尺寸）
2. 壓入轉動軸（H7/u6，建議縮配／熱脹法）→ **此步驟後若軸承座面有變形，需重新研磨**
3. 安裝 YRT100 軸承內圈（js5，若干涉請加熱至 80–100°C）
4. 裝配軸承座（H6），鎖附外圈螺栓（15 × M5，8.5 Nm）
5. 裝配工作台（H7/k6 過渡定位），鎖附內圈螺栓（16 × M5，8.5 Nm）及定位銷
