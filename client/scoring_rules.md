# 機台推薦評分規則 (Scoring Rules)

本文件描述 `company.html` 中用於篩選與評分機台的邏輯。

## 1. 硬性篩選 (Hard Filters)

只要不符合以下任一條件，機台會直接被**淘汰 (Rejected)**，分數雖會計算但狀態為 `ok = false`。

### 基本篩選
*   **公司 (Company)**: 若使用者指定特定品牌，非該品牌機台直接淘汰。
*   **製程 (Process)**: 若使用者指定特定製程 (e.g., VMC)，非該製程機台直接淘汰。

### 規格篩選 (若使用者有輸入)
*   **軸數 (Axes)**: 機台軸數 < 使用者需求。
*   **行程 (Travel X/Y/Z)**: 機台行程 < 使用者需求。
*   **轉速 (RPM)**: 機台最高轉速 < 使用者需求。
*   **刀庫 (Tool Capacity)**: 機台刀庫容量 < 使用者需求。
*   **棒材能力 (Bar Capacity)**: 機台棒材能力 < 使用者需求 (車床類)。
*   **工件尺寸 (Workpiece)**: 機台最大工件直徑/長度 < 使用者需求。

### 精度篩選 (Accuracy)
*   **重現性 (Repeatability)**:
    *   若機台**無重現性資料**，且使用者有輸入需求 -> **淘汰** (標記：缺精度資料)。
    *   若機台重現性數值 > 使用者需求 -> **淘汰** (標記：精度不足)。
    *   *(註：比較時皆以 mm 為單位)*
*   **定位精度 (Positioning)**:
    *   若機台**無定位精度資料**，且使用者有輸入需求 -> **淘汰**。
    *   若機台定位精度數值 > 使用者需求 -> **淘汰**。

---

## 2. 評分機制 (Scoring)

若機台通過所有篩選，則獲得基礎分數與加分。

### 基礎得分 (符合即得分)
*   **軸數**: 符合 +10 分 (每多一軸再 +1 分)。
*   **行程**: 符合各軸 +8 分 (每超過需求 200mm 再 +1 分，上限 +5)。
*   **轉速**: 符合 +8 分 (每超過需求 2000rpm 再 +1 分，上限 +8)。
*   **刀庫**: 符合 +6 分 (每超過需求 10 支再 +1 分，上限 +10)。
*   **棒材/工件**: 符合各項目 +6 分 (依據超過幅度微量加分)。

### 精度加分 (Accuracy Bonus)
*   **重現性**:
    *   符合需求 (+20 分)。
    *   **能力比 (Ratio)** Bonus:
        *   若 `使用者需求 / 機台能力 >= 2` (即機台精度是需求的兩倍好) -> 再 +10 分。
        *   若 `比率 >= 3` -> 再 +10 分。
*   **定位精度**:
    *   符合需求 (+15 分)。
    *   **能力比 (Ratio)** Bonus: `比率 >= 2` -> 再 +5 分。

---

## 3. 應用場景權重 (Application Scenarios)

根據使用者選擇的場景，調整分數或增加特殊標記。

### 模具加工 (Mold & Die)
*   **核心重點**: 極致的精度與表面品質。
*   **懲罰 (Penalty)**: 若機台**缺精度資料**，總分直接 **x 0.5** (減半)，並標記「風險極高」。
*   **獎勵 (Bonus)**:
    *   **超高精度**: 若重現性 ≦ 0.005mm (+30 分)。
    *   **高精度**: 若重現性 ≦ 0.010mm (+15 分)。
    *   **高轉速**: 若 RPM ≧ 15,000 (+10 分)。

### 航太加工 (Aerospace)
*   **核心重點**: 多軸、可靠性、大工件。
*   **標記**: 若缺精度資料，會標記警告，但暫未強制扣分 (仰賴信賴度指標)。

### 量產加工 (Mass Production)
*   **核心重點**: 效率 (Cycle Time)。
*   **獎勵**:
    *   **快送速度 (Rapid)**: 若 X 軸快送 ≧ 48 m/min (+10 分)。

---

## 4. 信賴度指標 (Confidence Score)

獨立於主分數之外，顯示機台資料的完整性 (0-100%)。

*   **基本資料**: 有轉速 (+20%)、有 X 行程 (+20%)、有刀庫資料 (+10%)。
*   **精度資料**:
    *   有重現性資料 (+30%)。
    *   有標註檢驗標準 (如 ISO/VDI/JIS) (+10%)。
    *   有定位精度資料 (+10%)。

> **注意**: 若信賴度過低，即使分數高，也代表該推薦結果可能基於缺漏的資料，需謹慎參考。
