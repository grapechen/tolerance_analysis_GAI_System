IF YOU GET ERRORS
─────────────────────────────────────────────────────────────────────────

Error: "Can't connect to MySQL server"
  → MySQL is not running
  → Fix: Start MySQL/MariaDB service first

Error: "ModuleNotFoundError: No module named 'OCC'"
  → pythonocc-core not installed
  → Fix: pip install pythonocc-core --only-binary :all:

Error: "table doesn't exist" in browser
  → Database tables not created
  → Fix: Run setup_database.py again

Error: "404 Not Found" on page load
  → Flask not running or port wrong
  → Fix: Check Flask console, restart if needed

Error: "Cannot find STEP file" or file access issues
  → File permissions problem
  → Fix: Check file permissions, run with appropriate privileges
