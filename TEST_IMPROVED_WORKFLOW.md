## ✅ 測試步驟

### 【步驟 1】上傳 STEP - 立即顯示 3D
```
預期行為：
□ 點擊「📤 上傳 STEP」
□ 選擇：c:\test0402\cpp_workspace\build\stepcode\Release\bin\part.stp
□ 提示：「✅ STEP 檔案已上傳，3D 模型已加載。」
□ 右側面板出現 3D 檢視器，顯示灰色半透明模型
□ 可以旋轉/縮放模型
□ 伺服器輸出：「[OK] STEP 已加載：XX 個面」

時間：3-5 秒
```

### 【步驟 2】上傳 XLSX
```
預期行為：
□ 點擊「📊 上傳 XLSX」
□ 選擇：c:\test0402\0405_test\2_壓縮公差標準代碼表.xlsx
  或其他 SFA XLSX 檔案
□ 提示：「✅ XLSX 檔案已上傳。現在點擊「比對 & 解析 PMI」進行比對。」

時間：< 1 秒
```

### 【步驟 3】點擊「比對 & 解析 PMI」
```
預期行為：
□ 進度條出現，顯示 6 個步驟
□ 每個步驟依序完成：
  1. [PARSING] 正在解析 Excel... (50/50)
  2. [LOADING] 正在準備 STEP 數據... (1/6)
  3. [LOADING] 正在加載 ASSOCIATION... (2/6)
  4. [LOADING] 正在建構備援數據... (3/6)
  5. [TESSELLATE] 正在解析 Tessellated 標註... (4/6)
  6. [WRITING] 正在寫入 N 個 PMI 項目... (5/6)

□ 進度條從 0% 逐漸增長到 100%
□ 完成後顯示：「✅ PMI 比對完成！• XX 個面 • YY 個 PMI 項目」

時間：3-8 秒
```

**伺服器控制台應顯示**：
```
[Reusing] 使用記憶體中已加載的 STEP（XX 個面）
[OK] 從 TAO Sheet 建立 N 條精確鏈結
[Hook] annotation_XX → TAO#N (dist=X.XXXmm)  <- Smart Proximity Hook
智能掛鉤：透過空間距離自動聯繫了 M 條導引線
```

### 【步驟 4】查看 PMI 清單
```
預期行為：
□ 右側面板下方出現 PMI 清單表格
□ 表格顯示所有已解析的項目，包含：
  - Label (標註名稱)
  - Type Code (類型)
  - Semantic ID (語義 ID)
  - TAO ID (標註 ID)
  - Face IDs (面列表)

□ 其中某些行顯示 semantic_id = "auto_pmi_N" 或 "unmapped_N"
  這表示由 Smart Proximity Hook 或備援機制建立的連結

□ 可以看到每個項目的完整信息
```

### 【步驟 5】交互式高亮 - 點擊 PMI 項目
```
預期行為：
□ 在 PMI 清單中點擊任一行
□ 3D 模型中對應的特徵面變成彩色（綠/紫/橙等）
□ 標註的導引線顯示為黑色直線
□ 相機自動聚焦到該特徵

□ 點擊不同的項目：
  ✓ 特徵面顏色改變
  ✓ 導引線位置改變
  ✓ 相機重新聚焦

示例交互序列：
  1. 點擊「annotation_01」→ 綠色面 + 導引線
  2. 點擊「annotation_02」→ 紫色面 + 不同的導引線
  3. 點擊「annotation_03」→ 橙色面 + 第三組導引線
```

### 【步驟 6】導出 CSV
```
預期行為：
□ 右側面板上方看到「📥 導出 CSV」按鈕
□ 點擊後，檔案自動下載：step_pmi_export.csv

□ 打開 CSV，驗證包含以下欄位：
  - label
  - type_code
  - semantic_id (包括 auto_pmi_N 項)
  - tao_id
  - face_ids
  - nominal_size
  - it_grade

□ 行數 = PMI 清單中的項目數
□ 所有 auto_pmi_N 項都包含在內
```

## 🔍 Smart Proximity Hook 驗證

在伺服器控制台尋找這些行：

```
[Hook] annotation_XX → TAO#N (dist=X.XXXmm)
[Hook] annotation_YY → TAO#M (dist=Y.YYYmm)
智能掛鉤：透過空間距離自動聯繫了 K 條導引線
```

**驗證項目**：
□ 至少有 1 個 [Hook] 訊息
□ dist 值都 < 2.0 (mm)
□ PMI 清單中看得到對應的 auto_pmi_N 項
□ 這些項的 TAO ID 與 [Hook] 訊息匹配

## ⚠️ 常見問題排除

| 問題 | 症狀 | 解決方案 |
|------|------|--------|
| 卡在「正在解析 Excel」| 進度條停留在 0% | 1. 檢查 XLSX 格式 2. 檢查伺服器日誌 |
| 卡在「正在三角化」| 進度條停留在 50-80% | 減少面數或增大 deflection 值 |
| 沒有 3D 模型出現 | 上傳 STEP 後面板空白 | 檢查 STEP 檔案有效性；清除瀏覽器快取 |
| 沒有 PMI 清單 | 解析完成但無列表 | XLSX 解析失敗；檢查欄位名稱 |
| 沒有導引線 | 高亮面出現但無線 | TAO 解析失敗；檢查 STEP 標註 |
| 導出 CSV 無數據 | CSV 是空檔案 | PMI 清單為空；重新執行步驟 3-4 |
